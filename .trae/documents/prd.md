# CryptoQuant Web 可视化界面产品需求文档

## 1. 产品概述

**产品名称：** CryptoQuant Web Dashboard  
**产品类型：** 加密货币量化交易回测结果可视化平台  
**核心功能：** 展示量化交易回测结果，提供交互式K线图表、持仓分析、交易记录查看等功能  
**目标用户：** 量化交易者、程序员、加密货币投资者

## 2. 功能需求

### 2.1 核心功能

#### 2.1.1 图表展示模块
- **K线图表**：显示历史价格走势，支持缩放、平移
- **技术指标叠加**：支持显示MA、RSI、MACD、Bollinger Bands等指标
- **交易信号标记**：在图表上标记买入卖出点
- **时间范围选择**：支持1分钟、5分钟、15分钟、1小时、4小时、1天等多种周期

#### 2.1.2 回测结果概览
- **关键指标卡片**：
  - 总收益率
  - 夏普比率
  - 最大回撤
  - 胜率
  - 总交易次数
  - 盈亏比
- **资金曲线图**：展示账户权益变化

#### 2.1.3 交易记录列表
- **交易列表**：显示每笔交易的详细信息
  - 交易时间
  - 交易方向（买入/卖出）
  - 交易价格
  - 交易数量
  - 盈亏金额
- **筛选功能**：按日期范围、交易方向筛选

#### 2.1.4 持仓管理
- **当前持仓显示**：展示持有的仓位
- **持仓明细**：买入均价、当前价格、盈亏情况

#### 2.1.5 策略选择器
- **策略列表**：MA交叉策略、RSI策略、MACD策略
- **参数配置**：短期周期、长期周期等参数
- **回测控制**：开始/停止回测、设置时间范围

### 2.2 用户交互

#### 2.2.1 图表交互
- 鼠标悬停显示K线详情（开盘价、最高价、最低价、收盘价、成交量）
- 鼠标滚轮缩放图表
- 拖拽平移图表
- 点击切换不同的时间段

#### 2.2.2 数据刷新
- 手动刷新按钮
- 自动定时刷新（可选）

### 2.3 响应式设计

#### 2.3.1 桌面端适配（PC）
- 三栏布局：
  - 左侧边栏：策略选择、参数配置
  - 中央主区域：K线图表
  - 右侧边栏：持仓信息、交易记录
- 最小宽度：1024px

#### 2.3.2 移动端适配（手机）
- 单栏布局：
  - 顶部：关键指标卡片
  - 中部：K线图表（可全屏）
  - 底部：Tab切换（持仓/交易/设置）
- 支持触摸操作：缩放、平移
- 最小宽度：320px

## 3. UI/UX 设计规范

### 3.1 视觉风格
- **主题**：深色主题，类似TradingView
- **配色方案**：
  - 背景色：#131722（深蓝黑色）
  - 卡片背景：#1e222d
  - 主色调：#2962FF（蓝色）
  - 上涨色：#26A69A（绿色）
  - 下跌色：#EF5350（红色）
  - 文字色：#D1D4DC（灰白色）
  - 次要文字：#787B86（灰色）

### 3.2 字体规范
- **主标题**：思源黑体 Bold, 20px
- **副标题**：思源黑体 Medium, 16px
- **正文**：思源黑体 Regular, 14px
- **数据**：DIN Alternate Bold, 18px
- **小字**：思源黑体 Light, 12px

### 3.3 间距规范
- **卡片内边距**：16px
- **卡片间距**：12px
- **组件间距**：8px

### 3.4 交互反馈
- 按钮悬停：背景色加深10%
- 卡片悬停：轻微上浮（translateY: -2px）
- 加载状态：骨架屏动画
- 成功/失败：Toast提示

## 4. 技术实现

### 4.1 技术栈
- **前端框架**：原生 HTML5 + CSS3 + JavaScript ES6+
- **图表库**：轻量级Canvas图表（自定义实现）
- **数据格式**：JSON

## 5. 性能要求

- 首屏加载时间：< 2秒
- 图表渲染帧率：60fps
- 支持数据点：最多10,000个K线
- 移动端流畅度：30fps

## 6. 兼容性

- **浏览器**：Chrome、Safari、Firefox、Edge（最新版）
- **操作系统**：Windows、macOS、iOS、Android
- **分辨率**：320px - 4K
